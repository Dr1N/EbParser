# EbParser
https://ebanoe.it/ parser.
Parse site to SQLite db
## Used
* .Net core v3
* AngleSharp
* CloudFlareUtilities
* EF Core
## Database tables
1. Tags
1. Posts
1. Comments
1. PostTags
1. Files
## Build
Build solution in VS2019 or via dotnet (v3+)
## Command line
* -f - save images from site (to /files folder)
